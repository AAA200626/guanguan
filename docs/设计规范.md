# 设计规范

> 创建日期：2026-07-18 | 状态：已确定

## 视觉风格

- **风格**：苹果极简风
- **配色**：马卡龙「薄荷青柠」色系
- **氛围**：清爽、治愈、轻量、可信赖

## 配色方案 — 薄荷青柠 🌿

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#7DCFB6` | 薄荷绿，按钮、进度条、强调 |
| 主色浅 | `#A8E6CF` | 卡片背景、选中态 |
| 主色超浅 | `#D6F5E8` | 页面底色 |
| 辅色暖调 | `#FFE8B3` | 杏色点缀（金币、奖章） |
| 卡片白 | `#FFFFFF` | 卡片底色 |
| 全局灰 | `#F5F5F5` | 全局背景 |
| 深文字 | `#2C3E50` | 标题、金额 |
| 浅文字 | `#95A5A6` | 辅助说明 |

> 公式：低饱和马卡龙主色(60%) + 白/灰背景(30%) + 高饱和点缀(10%)

## 字体层级

| 层级 | 大小 | 粗细 | 用途 |
|------|------|------|------|
| H1 | 32sp | Bold | 已存总金额 |
| H2 | 20sp | Bold | 页面标题 |
| H3 | 16sp | Medium | 卡片标题 |
| Body | 14sp | Regular | 正文 |
| Caption | 12sp | Regular | 辅助说明 |

字体：系统默认（Android: Roboto / iOS: SF Pro）

## 布局规范

- **卡片圆角**：16dp
- **卡片阴影**：elevation 2dp
- **卡片间距**：16dp
- **页面边距**：20dp
- **底部导航**：3 个 Tab（首页 / 挑战 / 我的）
- **FAB**：右下角薄荷绿浮动按钮

## 交互规范

- 打卡按钮：点击有缩放 + 薄荷绿波纹扩散动画
- 完成里程碑：全屏庆祝动画（金币飘落）
- 页面切换：Material 标准过渡（fade + slide）
- 删除确认：底部弹出确认 Sheet
- FAB 点击：展开快捷操作菜单
